# Rotrics.Net
SDK for controlling Rotrics robot arm.

## Notes

- At Y300, Z0, X seems to be safe for -250 to 250.
- At X0, Z0, Y seems to be safe for 180 to 390.
- At X0, Y300, Z seems to be safe for -110 to 160.
